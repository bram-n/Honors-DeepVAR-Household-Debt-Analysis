# A Deep VAR Analysis of Household Debt and GDP

## Installation
- Install Python 3.9
- Install Anaconda
- Ensure all the required packages listed in the `requirements.txt` are installed by using `conda install `
- Open ipynb files in VSCode or jupyter lab
